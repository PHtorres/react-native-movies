## Run Locally

Clone the project
```bash
  git clone https://github.com/PHtorres/react-native-movies.git
```


Install dependencies
```bash
  npm install
```

IOS extra step
```bash
  cd ios
  pod install
```

Android
```bash
  npx react-native run-android
```

IOS
```bash
  npx react-native run-ios
```
